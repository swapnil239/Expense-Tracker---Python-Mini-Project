# 🧾 Expense Tracker (Python Mini Project)

## Project Overview

This is a simple Expense Tracker Mini Project built in Python.

It allows users to record and manage their daily expenses in a structured way.  
The program runs in a loop and provides multiple options to interact with the expense records.

---

## Features

- Add a new expense with:
  - Date
  - Category
  - Description
  - Amount

- View all recorded expenses

- Calculate total expenses

- Exit the program

---

## How It Works

1. The program displays a menu with 4 options.
2. The user selects an option by entering a number.
3. When adding an expense, the details are stored in a list as a dictionary.
4. Users can view all stored expenses.
5. The total expense amount is calculated using a loop.
6. The program continues running until the user selects Exit.

---

## Data Storage Format

Each expense is stored in the following format:

{
  "date": "12-03-2026",
  "category": "Food",
  "description": "Lunch",
  "amount": 250.0
}

All expenses are stored inside a list.

1. Make sure Python is installed.
